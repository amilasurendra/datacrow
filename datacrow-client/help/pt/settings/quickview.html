Definições de vista rápida

Caminho de menu: Definições > Definições de vista rápida

Definições gerais




Se a opção "Mostrar imagens em separadores diferentes" estiver marcada cada imagem (se existentes) será mostrada num separador diferente, em vez de ser mostrada no separador Descrição.

Campos




Pode configurar os campos a mostrar na Vista rápida e a sua ordem, usando as definições da vista rápida. Clique num campo e use as setas verdes à direita na janela para o mover para cima, para baixo, para o início ou para o fim da lista.

Pode definir as seguintes propriedades para cada campo:
- Ativo: se um campo estiver ativo, é mostrado na vista rápida. Note que o campo só é mostrado se contiver um valor.
- Direção: determina onde o rótulo é posicionado em relação ao texto.
- Comprimento: define o comprimento máximo do valor no campo. A predefinição é 0, significando que o valor é mostrado sem alterações.

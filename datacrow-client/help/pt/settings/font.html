Letras

Caminho de menu: Definições > Definições > Letra

Pode definir a letra separadamente para o texto (os valores num item) e para os rótulos (os nomes de campo e outros textos do sistema). Pode escolher o tipo de letra, o tamanho e o estilo (negrito, itálico, simples).

Método de composição da letra

Esta parte é um bocadinho técnica. É importante que saiba que esta definição determina como as letras são mostradas no ecrã (compostas). Se tem um portátil, escolha um dos LCD da lista pendente.

Tente uma das definições e clique em Gravar para aplicar. Pode sempre voltar à predefinição se não gostar do resultado.

Definições de letras

No Data Crow são usados dois tipos de letra, um para os valores nos campos e outro para rótulos, botões, menus, etc.



Pode selecionar o tipo de letra, o tamanho e o estilo (negrito e/ou itálico). Um exemplo das definições é mostrado abaixo. Uma vez satisfeito com o resultado, clique em Gravar para aplicar as alterações.

Suporte para carateres asiáticos

O Data Crow suporta carateres chineses, japoneses e outros não ISO-8859-1 (suporta Unicode/UTF-8). Contudo, para trabalhar com estes carateres, tem de escolher um tipo de letra que também os suporte. Para sistemas Windows o melhor conjunto de carateres é aquele que tenha "Unicode" no nome, tal como "Arial Unicode MS".

Servidor Web

Caminho de menu: Servidor Web > Servidor Web

Ao usar este diálogo, pode configurar e iniciar o servidor Web. O diálogo Servidor Web pode ser aberto do menu de servidor Web.

Definições do servidor Web

Aqui pode definir qual a porta onde o servidor vai correr. O Data Crow não verifica se a porta está bloqueada por programas de segurança. Para isto, sugiro que consulte a documentação do seu sistema operativo e/ou a documentação do programa de segurança instalado.

Estado do servidor Web

Aqui é mostrado o estado atual do servidor. Pode iniciar o servidor clicando no botão azul "Reproduzir". O botão azul "Parar" pára o servidor, tornando a aplicação Web inalcançável para o mundo exterior.

Navegar até ao ambiente Web

Abra o ambiente Web premindo o botão "Ir para a aplicação Web" ou introduzindo o seguinte URL no seu navegador:
http://localhost:<port>/datacrow

Substitua a <porta> com o número de porta configurado. A primeira parte do URL é o endereço do servidor. Se executar localmente pode usar o seu localhost como endereço. Para o mundo exterior deve usar o endereço IP da sua máquina.

Segurança

O módulo Web usa os mesmos utilizadores que o programa Data Crow atual. Recomenda-se que crie utilizadores separados e que defina uma palavra-passe para cada um.

Configurar o ambiente Web

Os resumos dos itens podem ser configurados usando as "Definições de campos Web". Pode definir quais campos são mostrados e em que ordem. As páginas de detalhes do item dependem das predefinições. Por favor note que os campos mostrados também dependem dos direitos de acesso do utilizador.

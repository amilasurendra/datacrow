Onde começar?

Tudo bem.. Instalei o Data Crow e não tenho ideia por onde começar. Felizmente encontrei a ajuda.

Vou presumir que sabe que o Data Crow é um programa de gestão de coleções. No Data Crow pode registar os seus itens. Para permitir o registo de itens, o Data Crow vem com uma série de módulos, com Filmes, Programas e Livros. Encontrará estes módulos na lista de módulos, é impossível falhar.



A primeira coisa a fazer é abrir o módulo que quer usar. Simplesmente clique no ícone.

Abriu o módulo e está pronto a introduzir os seus itens.

Agora, como vai fazer isto depende de já ter ou não uma lista dos seus itens. Talvez esteja a usar outro produto, ou talvez tenha só um ficheiro de texto ou uma folha de cálculo com informação básica acerca dos seus itens.

Não tenho nada registado

No caso de estar a começar do zero, terá de criar os seus itens manualmente. Vamos, não tenha medo, não precisa de introduzir quase nada à mão!

Os meus itens são representados por ficheiros no meu sistema

Se quer criar itens dentro do Data Crow baseados em ficheiros no seu sistema, por exemplo filmes ou músicas, pode usar um Importador de ficheiros para criar itens do Data Crow baseados nestes ficheiros. O Data Crow vai extrair tanta informação quanto possível dos seus ficheiros.

Leia esta secção para mais informação sobre o Importador de ficheiros.

Os meus itens não são baseados em ficheiros

Se não tem ficheiros para importar pode tentar usar a procura online para encontrar informação sobre o seu item. Isto poupa-lhe uma quantidade de dactilografia! A procura online usa páginas (ou serviços como lhes chamamos) como a Amazon, Imdb.com e outros para encontrar informação e imagens para os seus itens.

Então, há um Assistente de item que faz uso da funcionalidade de procura online. Eu recomendo que comece aí. Clique em na barra de ferramentas. Este assistente existe para ajudar novos utilizadores a adicionar itens um por um.

Se se considera um utilizador proficiente também pode tentar a procura online diretamente. O benefício vem de poder adicionar múltiplos itens rapidamente. Leia esta secção para mais informação sobre a funcionalidade de procura online.

Não consegui encontrar informação acerca dos meus itens

Bem, então... começou do zero, os seus itens não são representados por ficheiros e a procura online não encontrou nada de útil para si. Terá de introduzir os itens manualmente.

Clique em na barra de ferramentas para adicionar um novo item. Isto vai abrir o Formulário de item. Introduza a informação manualmente neste formulário.

O Data Crow ainda o pode ajudar a tornar esta tarefa tão simples quanto possível.
Por exemplo, se tem ficheiros com muitos valores em comum pode criar um modelo com valores predefinidos. Os items baseados neste modelo recebem imediatamente estes valores. Leia mais sobre isto aqui.

Agora, se tem muitos itens para adicionar há uma última opção. Use a funcionalidade "Criar múltiplos items". Isto é explicado na secção Items da ajuda. Certifique-se que ativa Ver como tabela. Assim pode introduzir os itens como se fosse uma folha de cálculo.

Já registei os meus itens noutro programa

O Data Crow tem a capacidade de importar ficheiros de texto, tanto XML como CSV (ou outros com texto delimitado). Se registou os seus iten num formato de texto como o mencionado, ou se pode exportar a informação para um destes formatos, pode importar a informação diretamente para o Data Crow.

Terá de utilizar a funcionalidade Importar itens.

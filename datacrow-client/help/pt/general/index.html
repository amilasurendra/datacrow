Ajuda do Data Crow

Esta ajuda tenta guiá-lo na utilização do Data Crow. Esta página vai só falar genericamente acerca do programa. Para ajuda específica num assunto determinado, navegue pelo menu. Note que se premir F1 na área onde precisa de ajuda, o Data Crow abre o menu correspondente por si.

Uma introdução rápida

O Data Crow é um produto freeware que lhe permite manter as suas coleções. Como poderá ter lido na página do Data Crow, o Data Crow vem com um conjunto predefinido de módulos. Adicionalmente, o Data Crow permite-lhe criar os seus próprios módulos se precisar de algum (por exemplo gerir a sua coleção de selos) ou pode modificar os existentes para adicionar campos em falta.

Então o que significa o termo vago "módulo" no Data Crow? Um módulo é uma área auto-limitada no sistema onde pode administrar itens de um tipo determinado. Para explicar um bocadinho melhor, "Filmes" é um módulo e só permite a administração de filmes. Os programas não serão mostrados no módulo "Filmes"; estes residem no seu próprio módulo, "Programas".

Para cada módulo terá ferramentas específicas disponíveis. O módulo de filmes tem uma procura online disponível para procurar filmes no Imdb.com. Esta ferramenta não está disponível no módulo "Pessoas". Assim, vai encontrar ferramentas diferentes à sua disposição dentro de cada módulo.

A "Equipa"

Agora, vamos apresentar a equipa do Data Crow. O programador chefe é Robert Jan van der Waals, que é também o fundador deste produto, o designer, o funcionário da receção e o representante de vendas (fazendo um péssimo trabalho, já que o produto é grátis).

Histórico

Certamente que um produto desenvolvido por uma só pessoa não vai existir por muito tempo, pois não? Errado. O Data Crow é uma evolução de um produto chamado CD Library (também desenvolvido por mim, Robert Jan). O desenvolvimento desse produto começou em 2001. Em 2007 decidi reescrever todo o produto. Com o conjunto de funcionalidades expandido, incluindo a opção de criar os seus próprios módulos, o nome CD Library já não fazia justiça ao produto. A minha mulher surgiu com o nome Data Crow. Gostei dele (e não pude dizer Não... ainda não consigo) e aqui estamos.

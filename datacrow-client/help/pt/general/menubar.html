A barra de menu

A barra de menu contém ligações a todas as funcionalidades do Data Crow. Esta página de ajuda vai explicar-lhe o significado básico dos menus.

Ficheiro

Os items podem ser criados, apagados ou abertos. Pode também ver eventos ou sair da aplicação.
Assistente de item. O assistente de item é uma forma rápida de adicionar itens à base de dados (aparte a importação por ficheiros .csv ou XML). Começa por uma procura online (se o módulo suportar procuras online). Após o item ser encontrado pode editá-lo manualmente no passo seguinte. Quando terminar o assistente, o item é adicionado à base de dados.
Novo item <nome>. Com a opção de menu Novo item <nome> (o <nome> refere o nome do item, como Filme no módulo Filmes) pode adicionar manualmente um novo item ao Data Crow. Vai abri o Formulário de item.
Abrir item<nome>. Com a opção de menu Abrir item <nome> (o <nome> refere o nome do item, como Filme no módulo Filmes) pode abrir o item atualmente selecionado (no resumo) no Formulário de item. A informação do item não pode ser alterada usando esta opção, o item é aberto como só de leitura.
Editar item <nome>. Com a opção de menu Abrir item <nome> (o <nome> refere o nome do item, como Filme no módulo Filmes) pode abrir o item atualmente selecionado (no resumo) no Formulário de item. O formulário de item permite-lhe alterar a informação do item.
Gravar tudo. A vista Tabela permite-lhe editar os itens diretamente na vista. Para gravar todas as alterações que fez pode usar a opção de menu Gravar tudo.
Apagar. Apaga todos os itens selecionados (na vista) da base de dados. Esta ação não pode ser revertida (a não ser restaurando uma cópia de segurança). O sistema pede-lhe uma confirmação da ação.
Mostrar diário. O diário mostra eventos, coisas que aconteceram no sistema. Se alguma coisa correu mal, o Data Crow vai gravar a mensagem de erro aqui. É sempre um bom sítio para procurar se tiver problemas. Também mostra mensagens informativas , "Fiz isto aqui a esta hora".
O separador de informação de sistema mostra informação adicional. Que pastas estão a ser usadas, quais os recursos disponíveis, qual a versão Java, etc.
Sair. Sai da aplicação.

Administração

Manutenção. A maioria dos módulos usa listas pendentes ou campos multireferência. Um exemplo seria o campo Estado dos programas. Este campo Estado tem um conjunto predefinido de valores. Pode modificar a lista, completamente. A administração destes itens (itens de propriedade, parte do módulo de propriedade), pode ser feita através do menu Administração.
Administração de empréstimos. Este resumo dá-lhe uma imagem global dos seus empréstimos atuais, histórico dos empréstimos e itens com a devolução em falta. Tem opções de procura bastante poderosas com as quais pode trabalhar.

Nível de experiência

Principiante. Isto ativa o Data Crow para o modo Principiante, ocultando muitas das opções de vários menus.
Perito. Isto ativa o Data Crow para o modo Perito, dando acesso a todas as ferramentas e opções.

Módulos

Assistente Criar módulo. Permite-lhe criar um novo módulo. É uma ferramenta poderosa, permitindo-lhe desenhar a sua própria gestão de coleções de dentro da aplicação. Os items gravados por estes módulos são gravados na base de dados, tal como é feito como os módulos predefinidos. Terá muitas das ferramentas padrão à sua disposição ao gerir os seus itens. Há um manual específico, disponível para transferir na nossa página Web.
Assistente Copiar módulo. Este assistente permite-lhe criar um novo módulo baseado noutro já existente. Evita-lhe ter de definir todos os campos um por um, quando já existe um módulo que quase satisfaz as suas necessidades.
Assistente Alterar módulo. No caso de querer de querer adicionar um campo a um dos módulos existentes pode usar este assistente. Esta poderosa ferramenta permite-lhe alterar campos existentes ou adicionar novos campos. Adicionalmente, pode modificar algumas das definições (indicar se itens deste módulo podem ser emprestados a outras pessoas por exemplo).
Assistente Relações entre módulos. Uma ferramenta que lhe permite criar relações mãe-filho entre módulos. Pode encontrar um exemplo destas rela&ccedil;&otilde;es no m&oacute;dulo CDs de áudio. Este módulo é mãe do módulo de faixas do CD. Isto significa que não pode haver faixas de CD sem haver um CD de áudio. Não há forma de remover esta relação; tenha cuidado ao usar esta ferramenta!
Assistente Apagar módulo personalizado. Quer apagar um módulo? É este o assistente a usar. Em dois simples passos o módulo será destruído. Esta ação não pode ser desfeita. Note que também pode desativar módulos nas Definições Assim oculta o módulo sem o apagar. Esta opção não está disponível sem ter criado ou importado um módulo personalizado.
Assistente Exportar módulo. Então criou este espantoso módulo personalizado! Tanto trabalho e só está disponível para si. Porque não partilhá-lo com outras pessoas? Use este assistente para exportar a estrutura do módulo com todas as definições respetivas, dados e imagens. Esta opção não está disponível sem ter criado ou importado um módulo personalizado.
Assistente Importar módulo. Esta ferramenta permite-lhe importar um módulo previamente exportado com o Assistente de exportação de módulo. É um processo simples que vai carregar o módulo, os seus itens, definições e imagens.
Módulos ativos. Daqui alterna entre módulos. Simplesmente selecione o módulo para começar a utilizá-lo. Este menu está presente se desativou a lista de módulos, a área onde os módulos normalmente são selecionados.

Ver

Ver Cartão. O cartão é a vista mais atraente. Os items são mostrados como pequenos cartões e mostram uma imagem e uma curta descrição do item, baseada campos definidos como, "Descritivo?".
Ver Tabela. A vista tabela permite-lhe editar itens diretamente na vista, sem ter de abrir o formulário de item.
Alternar barra de filtragem rápida. Ativa ou desativa a barra de filtragem rápida.
Alternar lista de módulos. Ativa ou desativa a lista de módulos.
Alternar vista rápida. Ativa ou desativa a vista rápida.
Alternar painel de agrupamento. Ativa ou desativa o painel de agrumpamento.
Alternar barra de ferramentas. Ativa ou desativa a barra de ferramentas.

Filtrar

Filtrar. Aqui pode definir filtros complexos para encontrar os seus itens. Até os pode gravar, para que os possa usar no futuro.
Reaplicar filtro. Aplica o filtro novamente, atualizando completamente a vista.
Desfazer o filtro atual. Remove todos os filtros e carrega todos os itens.

Definições

Definições. A janela de Definições permite-lhe alterar todas as definições do sistema, tal com as de servidor proxy, associações de programas, opções de letra e muito muito mais. Estas definições são armazenadas por computador. Portanto, se tiver múltiplas instalações, em múltiplos computadores, mas partilha a mesma pasta de utilizador, estas definições não são partilhadas entre computadores. Tudo o resto é partilhado menos estas definições.
Definições gerais de campos. Não quer usar um certo campo? Quer alterar o rótulo de um dos campos? Pode encontrar aqui toda e qualquer opção de campos. Estas são definições muito importantes e é sempre bom dar uma vista de olhos a esta secção. A parte mais importante poderá ser a possibilidade de indicar como os seus campos são descritos com a opção "Descritivo?". Leia mais aqui.
Definições de vista rápida. A vista rápida pode ser adaptada aos seus desejos. Aqui pode indicar que campos são mostrados e em que ordem.
Definições de vista. Para modificar a vista atual, tal como as cores e a informação a mostrar, tem de usar as Definições de vista. As definições diferem ligeiramente entre ver Cartão e ver Tabela.
Definições de campos Web. Aqui pode alterar os campos como mostrado na aplicação Web do Data Crow.
Definições do formulário de item. Altere a aparência do Formulário de item. Até pode atribuir campos a separadores e alterar a ordem dos campos.
Aparência & utilização. Esta opção determina a aparência dos campos e outros objetos na aplicação. É melhor experimentar por si mesmo!
Editor de recursos. Qualquer rótulo pode ser alterado ou traduzido usando este editor. Pode até criar uma nova tradução.

Utilizador

Alterar palavra-passe. Altere aqui a sua própria palavra-passe.

Ferramentas

Importar ficheiros. Extrai informação de ficheiros no seu sistema. Por exemplo, para álbuns de música o Data Crow pode obter o nome do artista e do álbum do mp3 ou outros formatos de ficheiro. Adicionalmente, pode encontrar informação em falta usando um dos serviços online. Esta ferramenta só está disponível para módulo/itens "baseados" em ficheiros. Não deve ser confundido com a capacidade do Data Crow de importar itens de ficheiros de texto. Use Importar itens para importar itens de ficheiros CSV, texto ou XML.
Procura online. Encontre informação sobre os seus itens online, selecione o item e grave-o.
Nem todos os módulos têm esta ferramenta disponível. Módulos personalizados também não a terão a menos que tenha programado uma especificamente para o seu módulo.
Atualização em massa. Atualize muitos itens de uma só vez, usando as capacidades do Data Crow para procura online e importação de ficheiros.
Criar relatório. Crie relatórios HTML e PDF. Se esta opção de menu não estiver disponível, significa que não foram definidos ficheiros XSLT (modelos de relatório) para o seu módulo. Há um manual sobre como fazer isto na página do Data Crow.
Gráficos. Crie uma representação gráfica da informação dos seus itens.
Assistente de exportação de itens. Exporte os seus itens para um ficheiro CSV ou XML (recomendado).
Assistente Importar itens. Importe os seus itens de um ficheiro CSV ou XML.
Criar múltiplos itens. Permite a criação manual de múltiplos itens
Atualizar tudo. Atualize muitos itens (ou todos..) de uma só vez. Esta é uma ferramenta simples mas poderosa, aplicando valores introduzidos manualmente, a múltiplos itens em simultâneo.
Localizar & substituir. Substituir um valor por outro nos campos de texto. Vai-lhe mostrar uma pré-visualização (com sorte) antes de aplicar a substituição aos seus itens.
Autonumeração. Numere, ou renumere os seus itens. Todos e todos em simultâneo. Até tem opções para ignorar ou preencher espaços na numeração.
Reescrever título/nome. Reescreve os títulos dos seus itens. É uma excelente ferramenta para alterar títulos de "O Doutor" para "Doutor, O" O escritor de nomes é diferente para registos de pessoas em oposição a outros itens.
Cópia & restauro de seguranças. Crie uma cópia de segurança completa dos seus dados, incluindo definições, módulos e imagens. E também permite o restauro das cópias (estou só a dizer...).
Editor de base de dados. Acesso direto à base de dados. Ignora todas as regras de segurança que eu implementei na aplicação. Use por sua própria conta e risco! (faça.. cópias de segurança..)
Gestor de unidades. Procura nas suas unidades por ficheiros adicionados ao Data Crow e corrige as ligações aos ficheiros no caso de os ficheiros terem sido movidos ou renomeados. Uma ferramenta subvalorizada mas oh tão sofisticada.
Renomear ficheiros. Renomear ficheiros nas suas unidades baseado na informação presente no Data Crow. Também pode mover ficheiros para outra localização e nova estrutra de pastas. Mostra-lhe uma pré-visualização antes de efetuar qualquer ação.

Plugins

Este menu não está presente por predefinição. Só será mostrado se tiver instalado plugins personalizados.

Servidor Web

Este menu só aparece se tiver instalado a aplicação Web. Pode usar este menu para alterar as definições do servidor Web e iniciá-lo.

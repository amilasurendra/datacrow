A barra de ferramentas

A barra de ferramentas dá-lhe acesso rápido às opções mais importantes no Data Crow.



Segue-se uma curta descrição por ferramenta:

Assistente de item. Inicia um assistente para adicionar rapidamente um novo item.
Criar múltiplos itens. Permite a criação manual rápida de múltiplos itens
Criar. Com esta opção pode adicionar manualmente um novo item ao Data Crow. Vai abrir o Formulário de item.
Apagar. Apaga todos os itens selecionados (na vista) da base de dados. Esta ação não pde ser revertida (a não ser restaurando uma cópia de segurança).
Abrir. Abre o item atualmente selecionado na vista.
Gravar. A vista Tabela permite-lhe editar itens diretamente na vista. Use esta opção para gravar só o item selecionado.
Gravar tudo. A vista Tabela permite-lhe editar itens diretamente na vista. Use esta opção para gravar todas as alterações que fez.
Filtrar. A filtragem permite-lhe construir filtros complexos para encontrar os seus itens. Estes filtros podem ser gravados,para que os possa usar novamente no futuro.
Reaplicar o filtro. Aplica o filtro novamente, atualizando completamente a vista.
Procura online. Encontre informação sobre os seus itens online, selecione-a e grave-a.
Gráficos. Crie uma representação gráfica da informação dos seus itens.

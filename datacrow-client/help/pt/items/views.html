Vistas

Caminho de menu: Ver

Uma vista é uma área do Data Crow onde os itens são mostrados. Atualmente há duas vistas, a vista Tabela e a vista Cartão. As vistas podem diferir de módulo para módulo. O módulo contentor não dispões da vista tabela, por exemplo. Módulos personalizados têm, por predefinição, amboas as vistas disponíveis.

Barra de ferramentas

Vista rápida

A vista rápida fornece informação adicional sobre o seu item. Clique num item na vista e os seus detalhes serão mostrados na vista rápida.

Abrir um item

Abra um item com duplo clique na vista. O Data Crow abre o Formulário de item. O Data Crow abre o item em modo de edição ou só de leitura, conforme as suas definições.

Menu do clique com o botão direito

Quando clica com o botão direito, aparece um menu com as seguintes opções:


Abrir item de <nome>. Esta opção (<nome> significa o módulo em uso, como Filmes no módulo Filmes) abre o item atualmente selecionado (no resumo) no formulário de item. A informação do item não pode ser modificada com esta opção, o item é aberto como só de leitura.
Editar item de <nome>. Esta opção (<nome> significa o módulo em uso, como Filmes no módulo Filmes) abre o item atualmente selecionado (no resumo) no formulário de item. A informação do item pode ser modificada com esta opção.
Editar como novo. Cria um novo item baseado no item selecionado. O item selecionado é usado como modelo. O Formulário de item é aberto e preenchido com a informação do item selecionado. Após gravar e fechar o formulário, terá adicionado um novo item ao Data Crow.
Apagar. Apaga todos os itens selecionados da base de dados. Esta ação não pode ser desfeita (só restaurando uma cópia de segurança). O sistema pede-lhe uma confirmação primeiro.
Ficheiro. Quando tem um ficheiro ligado ao seu item, aparece a opção Ficheiro. Com esta pode deixar o Data Crow encontrar o seu ficheiro caso tenha sido movido ou renomeado. Utiliza a funcionalidade Gestor de unidades.
Assistente Exportar item. Inicia o assistente para exportar os itens selecionados para um ficheiro CSV ou XML.
Criar relatório. Cria um relatório em formato HTML ou PDF para os itens selecionados. Se esta opção não estiver disponível significa que não foram definidos ficheiros XSLT (modelos de relatório) para o seu módulo. Há um manual sobre como fazer isto, que pode ser encontrado na página do Data Crow.
Ordenar. Ordena os itens. A ordem é armazenada; após reiniciar o Data Crow, a ordem permanece a mesma. Pode ordenar os itens por ordem ascendente ou descendente e pode ordenar por múltiplos campos.
Gestão de empréstimos. Com esta opção pode gerir a informação de empréstimos do item selecionado.
Gestão. A maioria dos módulos usam listas pendentes e campos multireferência. Um exemplo seria o campo Estado dos itens Programas. Este campo tem um conjunto predefinido de valores. Pode modificar a lista, completamente. A gestão destes itens (chamados itens de propriedade, parte dos módulos de propriedade) pode ser feita aqui.
Definições de vista. Para modificar a vista atual, tal como as cores e a informação a mostrar, tem de usar esta opção. As definições diferem ligeiramente entre a vista Cartão e a vista Tabela.
Ler informação do ficheiro. Extrai informação do ficheiro selecionado e adiciona a informação ao item. O item passa então a estar ligado ao ficheiro. Para importar itens baseado na informação do ficheiro, pode usar a funcionalidade Importar ficheiros.
Atualizar tudo. Atualiza todos os itens selecionados. É uma ferramenta simples mas poderosa, aplicando valores introduzidos manualmente a múltiplos itens em simultâneo.
Localizar & substituir. Substituir um valor por outro em campos de texto. Mostra-lhe uma pré-visualização primeiro (com sorte) antes de aplicar a alteração aos itens selecionados.

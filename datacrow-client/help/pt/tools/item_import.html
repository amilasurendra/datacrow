Assistente Importar itens

Caminho de menu: Ferramentas > Assistente Importar itens

Este assistente permite-lhe importar itens a partir de um ficheiro externo. Os itens são importados diretamente para a base de dados. Antes de iniciar o assistente, terá de escolher o módulo para o qual quer importar itens.


Formatos

Selecione o tipo de ficheiro que quer importar. Para ficheiros XML selecione Importar ficheiros XML. Para ficheiros de texto selecione Importar ficheiros CSV. Um ficheiro CSV válido contém um item por linha. Os valores de cada um dos campos são separados por um caráter específico.

Configuração

Dependendo do formato escolhido, algumas das opções de configuração poderão não estar disponíveis.
- Conjunto de carateres: selecione o conjunto de carateres ou a codificação. Na maioria dos casos, UTF-8 será suficiente. Caso apareçam carateres estranhos numa cadeia, ou muitos pontos de interrogação, provavelmente tem de escolher outro conjunto de carateres. O Data Crow não consegue adivinhar a codificação do ficheiro a importar. Para saber qual é a codificação correta, aconselho consultar a documentação do programa que exportou o ficheiro. Note que as exportações XML do Data Crow são sempre feitas em UTF-8.
- Separador de valores: para importações de ficheiros CSV, tem de indicar qual o caráter que separa os valores de cada campo. Defina o separador como "TAB", caso tenha sido esse o usado.
- Tentar comparar itens importados: se marcada, o Data Crow vai tentar comparar os itens, para encontrar possíveis coincidências com itens existentes na base de dados. Se as encontrar, a informação dos dois itens será unida.

Mapear campos de dados




No caso de uma importação de ficheiro CSV, ser-lhe-á pedido que mapeie os campos de dados originais para os campos do Data Crow. Este é um passo crucial na importação. Caso veja só um campo, certifique-se que selecionou o separador de campos correto no passo anterior. Selecione o campo destino para cada um dos campos originais na tabela. O campo destino pode ser selecionado numa lista pendente.

Limitações

A importação CSV vai ignorar sempre a primeira linha, uma vez que espera que os nomes dos campos estejam nessa posição.

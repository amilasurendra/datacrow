Assistente Exportar itens

Caminho de menu: Ferramentas > Assistente Exportar itens

Este assistente permite-lhe exportar itens para um ficheiro externo. Pode exportar itens de todo e cada módulo do Data Crow. Antes de iniciar o assistente, tem de selecionar o módulo que quer usar.


Formatos




O assistente pode exportar itens para ficheiros de texto ou XML. XML é o formato preferido, uma vez que os itens serão armazenados numa estrutura hierárquica em vez de uma estrutura plana, de linha única. É também o formato preferido para trocar itens entre instalações do Data Crow. Também é o formato usado internamente pelo Data Crow para operações como exportação e importação de módulos.

A exportação para texto, exporta os itens para um ficheiro de texto. Cada linha representa um único item. Os valores são separados por um caráter, que pode definir na configuração do assistente.

Opções de configuração

Exportar ficheiro




Selecione o ficheiro destino (obrigatório).

Imagens

No segundo passo do assistente, pode optar por exportar as imagens. É recomendado, uma vez que as imagens são assim armazenadas numa subpasta da localização do ficheiro.
- Copiar imagens: copia as imagens para uma subpasta. Se optar por não exportar as imagens, será exportada uma ligação à localização original da imagem.
- Dimensionar imagens: pode optar por armazenar uma versão mais pequena da imagem original. Especifique a largura e altura. As proporções originais são mantidas para que não acabe com imagens distorcidas.

Comprimento do texto

Pode indicar o comprimento máximo do texto. O valor "0" significa que o texto não será cortado. O Data Crow tentará encontrar a posição mais simpática possível para cortar o texto. Por exemplo, a seguir a um ponto final ou a um caráter de fim de linha.

Limitações

1. A exportação de texto não consegue exportar relações entre os itens.
2. A exportação de texto remove carateres de nova linha dos valores. Cada linha do ficheiro de texto contém um único item.

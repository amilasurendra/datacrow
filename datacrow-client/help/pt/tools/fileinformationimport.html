Importar ficheiros <nome>

Caminho de menu: Ferramentas > Importar ficheiros <nome>

Usando a funcionalidade de importação pode rapidamente encontrar informação num CD-ROM ou no disco rígido. A funcionalidade Importar ficheiros <nome> encontra-se no menu Ferramentas. O nome corresponde ao nome do módulo, tal como Filmes ou Programas

Nota: esta ferramenta não deve ser confundida com a capacidade do Data Crow de importar itens de ficheiros de texto. Use o Assistente Importar itens para importar itens de ficheiros CSV, texto ou XML.

Pastas/Ficheiros

Aqui pode selecionar os ficheiros a importar.



Use as opções seguintes para incluir ficheiros para processamento:

- Selecione pasta: selecione uma pasta na qual o importador deve procurar ficheiros. Caso marque a caixa Recursivo, o importador procura na pasta selecionada e também nas subpastas. Clique em Iniciarpara começar a procura. Cada ficheiro encontrado é acrescentado à tabela por baixo.
- Selecione um ficheiro: selecione um ficheiro específico cuja informação quer analisar. O ficheiro selecionado é adicionado à tabela por baixo.

Todos os ficheiros encontrados são adicionados à tabela de ficheiros.



Cada um dos ficheiros marcados será processado quando a importação for iniciada.

Definições




Informação da multimédia

- Contentor: selecione o contentor para os itens importados. Um contentor pode ser qualquer coisa, desde uma estante de CDs a uma casa.
- Meio de armazenamento: opcionalmente, pode selecionar o meio de armazenamento para os dados a importar. Este é o suporte físico em que os dados são armazenados (CD, DVD, etc.).

Utilização das informações da pasta

Permite que a informação da pasta seja usada para preencher campos, tais como título, nome do álbum, artista (estes dois últimos só estão disponíveis para o módulo Álbum de música). As opções disponíveis variam conforme o módulo que esteja a usar.
- Usar nome da pasta como título: selecione o contentor para os itens importados. Um contentor pode ser qualquer coisa, desde uma estante de CDs a uma casa.

Opções para ficheiros de música
As opções seguintes só se aplicam a ficheiros de música (no módulo Álbuns de música).

- Não usar informação de pastas: usar só informação do ficheiro de música, como a informação da etiqueta de um ficheiro MP3.
- Usar nome da pasta como nome do álbum: a pasta em que a música está tem o nome do álbum (por exemplo ..\..\Abba Gold).
- Usar nome da pasta como nome de artista: a pasta em que a música está tem o nome do artista (por exemplo ..\..\Abba).
- Usar nome da pasta como nome de artista e a subpasta como nome de álbum: a pasta em que a música está tem o nome do artista e a subpasta tem o nome do álbum (por exemplo ..\Abba\Abba Gold).

Limpeza de títulos

Quando desativa a utilização da informação da pasta no separador Definições, o Data Crow tenta criar o título a partir do nome do ficheiro. Como pode imaginar, isto requer alguma limpeza. Para obter melhores resultados, pode especificar palavras a remover dos nomes de ficheiro, e/ou usar um padrão regex.



O Data Crow will primeiro remove as palvras deinidas e depois aplica o padrão regex.

Remover as seguintes palavras

A importação tenta obter o título do nome do ficheiro, se a informação não puder ser analisada no próprio ficheiro. Aqui pode introduzir palavras que devem ser removidas ao transformar um nome de ficheiro em título. A comparação não é sensível a maiúsculas/minúsculas.

Aplicar o seguinte padrão regex

Um padrão regex é uma espécie de linguagem de script que pode ser usada para analisar informação. Palavras ou carateres coincidentes serão removidos.

Procura online

Este separador só aparece se o módulo que está a usar suportar serviços online.



Pode usar um serviço online para encontrar informação adicional, tal como capas, por exemplo. Pode decidir se quer usar este serviço, marcando a caixa Usar serviços online para encontrar informação em falta. Se marcada, pode escolher o serviço a usar. Estas definições estão melhor explicadas aqui.

Arte local

Arte local são imagens, armazenadas no seu sistema, que pertencem ao item importado.



Em vez de confiar no Data Crow para encontrar imagens na internet, pode ser preferível usar estes ficheiros locais.

Procura recursiva na pasta

Permitir procurar arte local numa das subpastas do item atualmente importado.

Procurar arte usando as seguintes palavras-chave

Especifique as palavras-chave, separadas por vírgulas (,) que o nome de ficheiro de uma imagem deve conter para ser considerado arte local.

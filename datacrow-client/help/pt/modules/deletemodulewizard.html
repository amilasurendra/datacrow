Apagar módulos

Caminho de menu: Módulos > Assistente Apagar módulo personalizado

Permite-lhe apagar um módulo personalizado. Um módulo personalizado é um módulo criado usando o Assistente Criar módulo. Note que é sempre aconselhável fazer uma cópia de segurança do Data Crow antes de continuar, usando a funcionalidade "Cópia & restauro de seguranças".

Iniciar o assistente

Inicie o assistente selecionando Assistente apagar módulo personalizado no menu Módulos.

Selecionar um módulo para apagar

No primeiro passo do assistente, selecione o módulo que quer apagar, clicando no ícone respetivo.

Confirmar a eliminação

O segundo passo mostra-lhe informação sobre o que vai acontecer quando apagar o módulo selecionado. Outros módulos podem depender da existência deste módulo. Um bom exemplo seria um módulo de propriedade, Y, que foi usado por outro módulo, X. O módulo X tem uma lista pendente de itens pertencentes ao módulo Y. Quando apagar o módulo Y, a referência do módulo X ao módulo Y também é apagada.
Para apagar este módulo e todos os seus itens, clique em Terminar. Para interromper o assistente(a qualquer altura) clique em Cancelar.

Exportar um módulo

Caminho de menu: Módulos > Assistente Exportar módulo

Se criou um novo módulo e quer partilhá-lo com outros utilizadores, pode usar o assistente Exportar módulo. Este assistente pode ser encontrado no menu Módulos. No primeiro passo seleciona qual o módulo que quer exportar. No segundo passo configura o processo de exportação. O último passo é a exportação efetiva.

É exportado o seguinte

- O módulo principal (personalizado).
- Os módulos relacionados.
- Relatórios que criou (scripts XSLT, localizados na pasta de relatórios do Data Crow).
- Definições: todas as definições que fez para os módulos específicos são exportadas e usadas como predefinições.
- Opcionalmente, pode exportar os itens dos módulos principal e relacionados.

Opções

- Exportar os itens do módulo principal: o módulo principal é o módulo que selecionou no primeiro passo deste assistente. Marque esta caixa caso queira distribuir, não só a definição do módulo, mas também os itens que criou no módulo.
- Exportar os itens dos módulos relacionados: módulos relacionados são todos os módulos que pertencem ou estão relacionados com o módulo principal. Um módulo relacionado poderia ser, por exemplo, um módulo de categoria. Mesmo se escolheu não exportar os itens do módulo principal, pode ser benéfico exportar os itens dos módulos relacionados. Dá a oportunidade a outros utilizadores de preencher o módulo principal com os seus próprios itens ao mesmo tempo que utilizam propriedades criadas por si.

Limitação

- Módulos predefinidos, como Programas, não podem ser exportados. O asistente é só para módulos personalizados (criados com o Assistente Criar módulo).
- Só pode exportar módulos de nível superior. Os módulos relacionados com este módulo também serão exportados. Um módulo de nível superior é um módulo mostrado no painel de módulos, como por exemplo Programas ou Filmes.

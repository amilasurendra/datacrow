Importar um módulo

Caminho de menu: Módulos > Assistente Importar módulo

Este assistente pode ser encontrado no menu <b>Módulos</b> na janela principal e pode ser usado para importar um módulo exportado usando o Assistente Exportar módulo. O assistente é poderoso e, no entanto, muito fácil de usar. No primeiro passo seleciona o ficheiro comprimido (zip) que contém o módulo a importar. No passo seguinte e final, a importação tem início.

É importado o seguinte

- O módulo principal e os módulos relacionados.
- Relatórios criados especificamente para o módulo.
- As predefinições.
- Os itens pertencentes aos módulos.

Limitação

O Data Crow precisa de reiniciar para as alterações terem efeito.

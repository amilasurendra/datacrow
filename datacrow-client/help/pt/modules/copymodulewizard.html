Assistente Copiar módulo

Caminho de menu: Módulos > Assistente Copiar módulo

O assistente Copiar módulo pode ser encontrado no menu Módulos na janela principal.

Este assistente permite-lhe criar um novo módulo baseado num módulo existente. No primeiro passo do assistente ser-lhe-á pedido que indique em qual módulo quer basear o novo módulo. O módulo existente será usado como modelo. A informação do módulo, como seja o nome, ícones e campos, pode ser alterada nos próximos passos. Estes próximos passos são idênticos aos do Assistente Criar módulo.

Limitações

- As relações parentais não são copiadas. Por exemplo, se quiser copiar o módulo CD de áudio, terá de copiar também o módulo Faixas do CD. Após copiar ambos, pode voltar a criar a relação parental usando o Assistente Relacionar módulos.
- Os dados do módulo não são copiados. Contudo, pode exportar os itens do módulo original usando o Assistente Exportar item e importá-los para o módulo copiado com o Assistente Importar itens.

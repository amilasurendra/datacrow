Relacionar módulos

Caminho de menu: Módulos > Assistente Relacionar módulos

Na barra de menu da janela principal vai encontrar um menu chamdo Módulos. Este menu contém assistentes para personalizar, criar e relacionar módulos, entre outras opções. Neste capítulo é explicado como relacionar dois módulos. Usando este assistente pode criar uma relação mãe-filho entre dois módulos. A relação entre CDs de áudio e faixas do CD é um bom exemplo destas relações. O CD de aúdio é o módulo-mãe e as faixas do CD são itens do módulo-filho. As faixas não podem existir sem um CD.

Iniciar o assistente

Inicie o assistente selecionando Assistente Relacionar módulos no menu Módulos.

Selecionar o módulo-mãe

No primeiro passo seleciona o módulo-mãe.

Selecionar o módulo filho

No último passo seleciona o módulo-filho.

Terminar o assistente

Depois de fornecer toda a informação, clique em Terminar ou em Terminar e novo. O primeiro termina o processo. Tem de reiniciar o Data Crow para que as alterações tenham efeito. O segundo termina o processo e reinicia o assistente (o que pode ser útil se quer criar mais relações).

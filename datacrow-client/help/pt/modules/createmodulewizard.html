Criar um novo módulo

Caminho de menu: Módulos > Assistente Novo módulo

Na janela principal do Data Crow encontra um menu chamado Módulos. Este menu contém assistentes para personalizar, criar e relacionar módulos. Neste capítulo vamos mergulhar na criação do seu próprio módulo.

Iniciar o assistente

Inicie o assistente selecionando Assistente Criar módulo no menu Módulos.

Selecionar o tipo de módulo

Há quatro tipos de módulo à escolha; propriedade, multimédia, outro e pessoas.

Módulos de propriedade

Este é um módulo muito simples. Tem dois campos importantes predefinididos: "nome" e "ícone". Este módulo é usado por outros módulos para criar itens de propriedade como géneros, categorias, meios de armazenamento, etc. Poderá ler mais sobre isto quando começar a definir campos para o seu módulo.

Módulos multimédia

Este tipo de módulo é muito comum no Data Crow. Programas, CDs de áudio e Filmes são exemplosde módulos multimédia. Este tipo de módulo tem um conjunto predefinido de campos comum a todos os tipos de multimédia: título, descrição, ano, localização, classificação e utilizador.

Outros módulos

Se nenhum dos acima lhe serve, escolha este tipo de módulo. Não tem campos predefinidos (exceto o campo ID, que é obrigatório para todos os módulos), tornando-o o perfeito candidato para criar um módulo de raíz.

Módulos de pessoas

Quer um módulo representando pesoas? Campos predefinidos: nome, nome próprio, apelido, fotografia, página web.

Preencher a informação básica

Neste passo vai dar nome ao módulo e aos seus itens, selecionar os ícones e introduzir uma descrição. Os nomes do módulo e seus itens não podem ser alterados, portanto escolha com cuidado.
- Descrição: uma descrição genérica do módulo, mostrada na barra de módulos ao passar o rato por cima.
- Nome do módulo: o nome do módulo, por exemplo, Filmes.
- Nome do item: o nome dos itens do módulo, por exemplo, Filme.
- Plural dos itens: o plural do nome de item, por exemplo, Filmes.


Clique nos ícones para os alterar. O maior é usado na barra de módulos (lado esquerdo da janela), o mais pequeno em toda a aplicação.

Estão disponíveis as seguintes opções (note que as opções variam de módulo para módulo):
- Os items podem ser atribuídos a um contentor: ativar esta opção torna disponível uma lista pendente de contentores para os itens do módulo, o que permite armazenar os itens em contentores. Um contentor pode ser uma estante, uma caixa de CDs ou até mesmo uma casa e respetivas divisões.
- Os itens representam ficheiros reais: se ativada são adicionados campos de ficheiro aos itens do módulo, tornando possível que os seus itens representem ficheiros no sistema.
- Os itens podem ser emprestados a outras pessoas: pode indicar se os itens do módulo podem ser emprestados a outras pessoas (não disponível para módulos de propriedade). Ativar esta opção disponibiliza a funcionalidade de empréstimos. Pode alterar esta informação mais tarde (Assistente Alterar módulo).

Definir campos

Veja "Definir campos".

Terminar o assistente

Após ter fornecido toda a informação clique em Terminar ou em Terminar & novo. O primeiro termina o processo de criação de módulo (tem de reiniciar o Data Crow antes de poder usar o módulo). O segundo termina o processo de criação de módulo e reinicia o assistente (o que pode ser útil quando quer criar mais do que um módulo).
